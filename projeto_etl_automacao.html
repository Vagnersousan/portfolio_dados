Solução de Automação Operacional (ETL Simples)

Eliminando o trabalho manual repetitivo com criatividade e tecnologia.

ETL & Automação JavaScript (Vanilla) Eficiência Operacional

O Desafio e a Virada de Chave

Durante meu estágio na TIM, identifiquei um processo crítico que estava consumindo recursos valiosos da equipe:

A rotina de consolidação de dados de servidores e aplicações dependia de mais de 30 planilhas enviadas por e-mail. O resultado era um processo de ETL (Extração, Transformação e Carga) manual que gerava:

  • Horas de trabalho manual perdidas semanalmente.
  • Inconsistências de dados e erros humanos.
  • Inviabilidade na manutenção de relatórios complexos.

A Solução Criativa

Em vez de insistir em fórmulas complexas no Power Query ou Excel, a 'virada de chave' foi buscar uma automação focada em simplicidade e eficiência. Utilizei conhecimentos de desenvolvimento web para criar um sistema local (HTML, CSS e JavaScript) que atua como middleware.

Como funciona o fluxo:
  • Entrada: O sistema recebe múltiplos arquivos (planilhas) via upload simultâneo.
  • Processamento: O JavaScript processa, consolida e remove duplicatas automaticamente seguindo as regras de negócio.
  • Saída: Gera um arquivo final limpo, padronizado e pronto para análise.

Impacto no Negócio

-90%

Redução no Tempo Operacional

Economia drástica de esforço manual na tarefa de consolidação.

100%

Foco Estratégico

Liberação da equipe para atuar na análise, e não na cópia de dados.

Este projeto provou que ser um Analista de Dados é ter a visão estratégica para identificar a melhor solução, usando as ferramentas como meio, e não como um fim.
